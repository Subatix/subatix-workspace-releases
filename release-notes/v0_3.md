# Subatix v0.3.0 - Initial Release

**Release Date:** January 2026

This is the initial public release of Subatix - an AI-native workspace for modern consulting work.

Built on the VS Code 1.106 foundation, Subatix brings AI capabilities directly into your workspace, alongside the vast VS Code infrastructure.

## New Features

### Subatix Agent
The integrated AI assistant built directly into the application:

- AI-powered workspace companion accessible from the sidebar
- User authentication system for accessing agent features
- Persistent workspace memory that remembers your project context
- Custom interface for seamless AI interactions

### Built-in Extensions

**Office File Viewer**
- View Excel spreadsheets (.xlsx, .xls)
- View Word documents (.docx, .doc)
- View PowerPoint presentations (.pptx, .ppt)
- Open office files directly in the workspace without external applications

**Subatix Themes**
- Subatix Dark - Professional dark theme for extended work sessions
- Subatix Light - Clean light theme for daytime use

### Workspace Features

**File Management**
- Multi-folder workspace support
- Advanced search across all project files
- Side-by-side file comparison
- Integrated file explorer

**Editing**
- Multi-cursor editing
- Find and replace across multiple files
- Syntax highlighting for common formats
- Auto-save functionality

**Extensibility**
- Install additional tools from the extension marketplace
- Customize your workspace to match your workflow

## Platform Support

**macOS (Apple Silicon)**
- Signed and notarized for Gatekeeper

**Windows (x64)**
- Signed with OV code signing certificate

## Installation

Download Subatix from the official website: [subatix.com](https://subatix.com)

## Getting Started

After installation:
1. Launch Subatix
2. Open the Subatix Agent panel from the sidebar
3. Create an account or sign in
4. Start working with AI assistance in your projects
